# ETRS-013
## Application en ligne qui permet de planifier un voyage en véhicule électrique.

L’utilisateur sélectionne d’abord son véhicule dans une liste, puis saisit un trajet entre deux
villes. L’application analyse alors l’autonomie du véhicule et les distances à parcourir. 

Elle génère ensuite un itinéraire détaillé indiquant les villes dans lesquelles l’utilisateur devra
s’arrêter pour recharger son véhicule, ainsi que le nombre d’arrêts nécessaires et leur position
optimale afin de réaliser l’intégralité du trajet en toute sérénité.
